# Conversor

Conversor es un módulo que nos sirve para hacer la conversión de Libras a Kg y viceversa. Es la primera tarea del [Curso de Fundamentos de Javascript](https://platzi.com/js) de [Platzi](https://platzi.com/).

## Descripción de La Conversión

- Ingresar un número en la caja del conversor de Libra a Kg para convertir Libras a Kg.
- Ingresar un número en la caja del conversor de Kg a Libra para convertir Kg a Libras.


## Instalación

```
npm install conversor
```

## Uso

```
import conversor from 'convertirLiKg'
import conversor from 'convertirKgLi'

//convertirLiKg() Convertir Libra a Kg
//convertirKgLi() Convertir Kg a Libra
```
